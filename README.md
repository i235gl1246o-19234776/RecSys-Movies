# Movie Recommendation System

Проект по разработке и сравнению алгоритмов рекомендательных систем на датасете **MovieLens**. Включает полный пайплайн: от EDA и генерации признаков до ML-ранжирования и подготовки артефактов для инференса.

## Реализованные подходы
* **Baseline** — предсказание на основе статистик.
* **Collaborative Filtering** — User-Based и Item-Based (косинусное сходство).
* **Matrix Factorization** — SVD (библиотека `surprise`).
* **ML-Ranking** — CatBoost Ranker (оптимизация `YetiRank` / `QueryRMSE`).

## Ключевые особенности пайплайна
* **Time-Based Split:** Критически важное разделение по временной метке (защита от утечки будущего).
* **Feature Engineering:** Мета-признаки (год, кол-во жанров), One-Hot Encoding, локальные и глобальные статистики.
* **Cold Start:** Стратегия обработки новых пользователей и фильмов через глобальные средние.
* **Interpretability:** Анализ модели и влияния признаков с помощью `SHAP`.
* **Inference Pipeline:** Генерация `Top-N` рекомендаций с исключением уже просмотренных фильмов.

* **NDCG** — оценка качества ранжирования (правильность сортировки фильмов).
